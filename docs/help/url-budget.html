üõ°Ô∏è Guardrails ‚Äî How to NOT Go Over Budget

1. Set a hard spending limit on your API account

Go to your provider's dashboard and set a monthly spending cap:

  • Anthropic: console.anthropic.com/settings/limits ‚Üí Set "Monthly spend limit" to $50 (per pay period)
  • OpenAI: platform.openai.com ‚Üí Settings ‚Üí Limits
‚úÖ This is your #1 guardrail. The API literally stops working when you hit the cap. No surprises.

2. Use Sonnet as your default model

Set Sonnet (not Opus) as your default so you're not accidentally burning premium tokens on "hey what time is it":

In your OpenClaw config:

"primary": "anthropic/claude-sonnet-4-20250514"

3. Enable prompt caching

This caches the system prompt and context so you're not paying full price every message. Saves 50-90% on input tokens.

4. Enable compaction

Auto-summarizes old conversation so context doesn't grow forever:

"compaction": { "mode": "safeguard" }

5. Start fresh conversations regularly

Use /new to start a new session when you switch topics. A 5-message conversation costs a fraction of a 100-message one.

6. Monitor your spending

Check your API dashboard regularly. Both Anthropic and OpenAI show daily/weekly spend graphs.
